## 🛠️ 安装方法

1. 下载并解压安装包
2. 打开浏览器，访问 `chrome://extensions/`
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择解压后的 `extension` 文件夹
6. 插件安装完成！

---

## 📝 使用说明

1. 点击浏览器右上角的插件图标，打开弹窗
2. 点击「上传简历」，选择你的 PDF/DOCX 简历文件
3. 等待 AI 解析完成
4. 访问任意招聘网站的申请表单页面
5. 点击页面上的悬浮按钮「🚀 一键填写」
6. 享受自动化填写！

 点击「激活」，即可解锁无限次使用
